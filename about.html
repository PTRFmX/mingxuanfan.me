About

This website aims at providing some basic knowledge of cyber security for any visitors who are unfamiliar with but interested in computer networks and security. Topics covered includes the introduction to computer networks, examples of common cyberattack categories, and the corresponding security measures.

The section of computer networks introduction will explain what is a network, how it relates to cyber spaces and cyber security issues. The section of common cyberattack categories introduces three common types of malicious attacks: The MITM attack, the DDoS attack, and attacks towards general vulnerabilities. The last section provides introductions to how modern security approaches deal with the previously mentioned attacks.
